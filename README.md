# 🚀 Тестовое задание: Верстка лендинга

Данный проект выполнен в рамках отбора на стажировку. Основная цель — создать современный, быстрый и адаптивный интерфейс с использованием продвинутых возможностей CSS.

## 🛠 Стек технологий
*   **Vite** — быстрая сборка проекта.
*   **SCSS** — модульная и понятная структура стилей.
*   **Swiper.js** — интерактивные слайдеры с курсами.
*   **Scroll Snapping** — эффект «примагничивания» секций при прокрутке.


## 🚀 Как запустить проект

Для запуска понадобится установленная **Node.js**.

1.  **Установи зависимости:**
    ```bash
    npm install
    ```

2.  **Запусти сервер для разработки:**
    ```bash
    npm run dev
    ```
    *После запуска проект будет доступен по адресу, указанному в терминале (обычно `http://localhost:5175`).*
